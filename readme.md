# Segmentador Automático

### Requerimos

1.  Cover e00 de cada localidad a segmentar, pxrad y  la C1.

2.  Programa Qgis 3.16 instalado.

3.  Buscador Firefox, Chromium, GoogleChrome

4.  Muchas buena onda : D
